# SwapOS

SwapOS is an experimental open-source operating system project created by Swapnil Bambole.

## Goals
- Build a lightweight operating system
- Create a custom kernel
- Develop a modern user interface
- Support future mobile and desktop devices

## Project Structure

- boot/ - Boot loader
- kernel/ - Kernel source
- system/ - System initialization
- drivers/ - Device drivers
- apps/ - Built-in applications
- framework/ - System framework
- docs/ - Documentation
